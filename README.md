DVWA Web Application Security Testing Lab

Project Overview

This project demonstrates the setup of a penetration testing lab using DVWA (Damn Vulnerable Web Application) and the execution of security testing activities in a controlled environment.

Lab Environment

- Hypervisor: VMware Workstation
- Attacker Machine: Kali Linux
- Target Machine: DVWA
- Network Configuration: Host-Only Network

Objectives

- Install and configure DVWA.
- Perform security testing in a safe environment.
- Identify common web application vulnerabilities.
- Document testing procedures and findings.

Tested Vulnerabilities

SQL Injection

- Tested input fields for SQL Injection vulnerabilities.
- Observed database information disclosure under vulnerable configurations.

Cross-Site Scripting (XSS)

- Tested reflected XSS vulnerabilities.
- Demonstrated execution of client-side scripts.

Project Structure

- Screenshots/ : Contains setup and testing screenshots.
- Reports/ : Contains the final project report.
- Scripts/ : Contains any supporting scripts used during testing.

Disclaimer

This project was conducted in a controlled laboratory environment for educational purposes only. All testing activities were performed on intentionally vulnerable systems owned by the project environment.
